🚀 Portfolio Website

A personal full-stack portfolio website built using Node.js, Express, MongoDB, and EJS.
This project showcases my skills, projects, and contact information with an admin dashboard to manage projects dynamically.


---

🌐 Live Demo

👉 https://portfolio-website-s81p.onrender.com


---

✨ Features

Home page with introduction

About section

Skills display

Dynamic Projects page (MongoDB integrated)

Admin dashboard to add projects

Contact page

Responsive design

Backend using Express.js

Database using MongoDB Atlas



---

🛠️ Tech Stack

HTML, CSS, JavaScript

Node.js

Express.js

MongoDB Atlas

Mongoose

EJS Templates

⚙️ Setup Instructions (Run Locally)

1. Clone the repository

git clone https://github.com/your-username/Portfolio-Website.git

2. Install dependencies

npm install

3. Create .env file

MONGO_URI=your_mongodb_connection_string

4. Run the server

node server.js

5. Open in browser

http://localhost:3000
